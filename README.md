# 搜读

一个神奇的网站！

## 功能说明

- [ ] 文档上传
- [ ] 数据提取、转录（文生图、文生音）
- [ ] 基于语义的关键词搜索，支持内容以音频形式播放
- [ ] 语音对聊（基于现有知识库）

## 部署和运行

### 本地

基于开源的模型，以`onnx`运行时为载体，根据不同需要加载不同模型。一方面可以有效利用本机性能，另外也便于开发调试

### 生产

由于模型文件通常比较大，远程部署分为两块：

> 优先serverless api，其次端侧兜底

- Web端：serverless api(openai | doubao)
- 客户端：[端侧部署](https://github.com/mlc-ai/mlc-llm)
